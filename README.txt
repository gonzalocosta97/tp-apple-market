El presente trabajo práctico consta de una tienda ficticia de productos (celulares) Apple.

Aclaraciones:

1) De acuerdo con las consignas del mismo, el sitio web fue desarrollado utilizando técnicas de maquetado "antiguas" (maquetado con floats). No era posible, por ejemplo, realizarlo con Flexbox o Grid.

2) Posee responsive design (para tablets y smartphones).

3) Se utiliza JavaScript para la generación de contenido dinámico y para dotar de interactividad algunas secciones de la web (por ejemplo, el cálculo de garantía).

4) Se utiliza la biblioteca Lightbox (jQuery) en la sección de productos (para la visualización de las imágenes de los mismos).
